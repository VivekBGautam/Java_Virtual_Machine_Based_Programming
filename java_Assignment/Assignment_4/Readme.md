# Assignment 4 – Warmup Programs (Java Version)

This folder contains the warmup Java programs related to factors and non-factors of a number.
Each program performs a specific operation as described below.

Programs Included

Program4_1.java

Multiplies all the factors of a given number.

Program4_2.java

Displays all factors of a number in reverse order.

Program4_3.java

Displays all non-factors of a number.

Program4_4.java

Displays Summation all non-factors of a number.

Program4_5.java

Displays the difference between
(sum of factors) − (sum of non-factors).

How to Compile and Run (Java)

Use the following commands in the terminal:

Compile
javac filename.java

Run
java filename


Example:

javac Program4_1.java
java Program4_1
