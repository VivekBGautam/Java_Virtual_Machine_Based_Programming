# Assignment 03

This folder contains my warm-up Java programs.
Each program performs a specific fundamental operation useful for building programming logic.

Programs Included

Program3_1.java
Prints the given number of even numbers on the screen.

Program3_2.java
Displays all the factors of a given number.

Program3_3.java
Displays only the even factors of a given number.

Program3_4.java
Converts a given character from uppercase to lowercase and vice-versa.

Program3_5.java
Checks whether the given input character is a vowel or not.

How to Compile and Run (Java)

Use the following commands in the terminal:

javac filename.java
java filename


Example:

javac Program3_1.java
java Program3_1
